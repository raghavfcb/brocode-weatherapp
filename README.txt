 This is a sleek and minimal desktop weather application built in Python using the OpenWeatherMap API and PyQt5 for a responsive GUI. It displays current weather condition and temperaturee at the specified location.

Features of this app:
-- Real-time weather data from OpenWeatherMap
-- Search by city name
-- Displays Temperature (°C/°F and Weather condition (e.g., Clear, Rain, Mist)
-- Clean and responsive PyQt5 GUI
-- Error handling for invalid city names or API issues


Tech Stack used:
- Python 3.x
- PyQt5
- Requests (to fetch API data)
- OpenWeatherMap API
